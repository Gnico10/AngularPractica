{{ nombre }} uppercase {{ nombre | uppercase }} Todo a mayúsculas.
{{ nombre }} lowercase {{ nombre | lowercase }} Todo a minúsculas.
{{ nombre2 }} Capitalizado con variable por defecto. ({{ isCapitalizado }})
Cambiar 		{{ nombre2 | capitalizado: isCapitalizado }} Pasar texto a capitalizado. Debe agregarse el pipe en app.modules.
{{ nombre }} slice:3 {{ nombre | slice: 3 }} Cortar los primeros 3 caracteres.
{{ nombre }} slice: 0:3 {{ nombre | slice: 1:3 }} Cortar desde la posición 0 hasta la posición 3.
{{ arreglo }} slice: 1:5 {{ arreglo | slice: 1:5 }} Cortar desde la posición 1 hasta la posición 5 de un arreglo.
{{ PI }} number {{ PI | number }} Redondea el número decimal a un formato de 0,000.
{{ PI }} number: '1.0-0' {{ PI | number: "1.0-0" }} Extrae la parte entera de un decimal. 0-0 especifica que no se quieren decimales.
{{ PI }} number: '3.0-0' {{ PI | number: "3.0-0" }} Extrae 3 números en la parte entera.
{{ PI }} number: '.0-2' {{ PI | number: ".0-2" }} Todos los enteros, pero 2 números en los decimales.
{{ porcentaje }} percent {{ porcentaje | percent }} Número decimal a formato de porcentaje. (Redondea).
{{ porcentaje }} percent: '2.2-2' {{ porcentaje | percent: "2.2-2" }} Muestra dos posiciones enteras y dos decimales.
{{ salario }} currency {{ salario | currency }} Muestra el valor en un formato de moneda.
{{ salario }} currency: 'EUR' {{ salario | currency: "EUR" }} Muestra el valor junto al simbolo de moneda. (ISO 4217).
{{ salario }} currency:'CAD':'symbol':'.0-0' {{ salario | currency: "CAD":"symbol":".0-0" }} Muestra el formato de la divisa completa y quita los decimales.
{{ salario }} currency:'CAD':'symbol-narrow':'.0-0' {{ salario | currency: "CAD":"symbol-narrow":".0-0" }} Quita el formato de la divisa completa y quita los decimales.
{{ valorPromesa }} async {{ valorPromesa | async }} Espera a que la promesa responda y luego coloca el valor devuelto.
{{ fecha }} date {{ fecha | date }} Formatea un objeto Date.
{{ fecha }} date:'medium' {{ fecha | date: "medium" }} Coloca un formato de fecha predeterminado que muestra la hora (12hs).
{{ fecha }} date:'short' {{ fecha | date: "short" }} Coloca un formato de fecha predeterminado corto.
{{ fecha }} date:'MMMM - dd' {{ fecha | date: "MMMM - dd" }} Coloca un formato de de mes y día.
{{ fecha }} date:'MMMM - dd':'':'{{ idioma }}'
{{ fecha | date: "MMMM - dd":"":idioma }} Cambiar el idioma. El idioma debe registrarse antes de poder ser usado.
